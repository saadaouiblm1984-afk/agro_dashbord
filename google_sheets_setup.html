إعداد Google Sheets

اتبع هذه الخطوات لتوصيل لوحة التحكم بـ Google Sheets

1

إنشاء Google Sheets

أولاً، قم بإنشاء جدول Google Sheets جديد:

  1. اذهب إلى Google Sheets
  2. أنشئ جدول جديد باسم "Admin Dashboard Database"
  3. أنشئ الأوراق التالية بالأسماء المحددة:
products orders categories customers order_items promotions
ملاحظة: يجب أن تكون أسماء الأوراق مطابقة تماماً للأسماء المذكورة أعلاه.
2

إعداد الأعمدة

أضف الأعمدة التالية لكل ورقة:

ورقة products:
A1: product_id B1: category_id C1: product_name D1: qtepack E1: price F1: image_url G1: status H1: description
ورقة orders:
A1: order_id B1: customer_id C1: customer_name D1: phone E1: address F1: total_price G1: order_status H1: order_date I1: items
ورقة categories:
A1: category_id B1: category_name C1: image_url
3

الحصول على معرف الجدول (Spreadsheet ID)

احصل على معرف الجدول من رابط Google Sheets:

https://docs.google.com/spreadsheets/d/[SPREADSHEET_ID]/edit

مثال:

https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms/edit

المعرف هو:

1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms
4

إنشاء Google Apps Script

إنشاء سكريبت للتعامل مع البيانات:

  1. في جدول Google Sheets، اذهب إلى: Extensions > Apps Script
  2. احذف الكود الافتراضي
  3. ألصق الكود التالي:
function doGet(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet(); const action = e.parameter.action; try { switch(action) { case 'getProducts': return getSheetData('products'); case 'getOrders': return getSheetData('orders'); case 'getCategories': return getSheetData('categories'); case 'getCustomers': return getSheetData('customers'); case 'getOrderItems': return getSheetData('order_items'); case 'getPromotions': return getSheetData('promotions'); default: return ContentService.createTextOutput('Invalid action') .setMimeType(ContentService.MimeType.TEXT); } } catch(error) { return ContentService.createTextOutput('Error: ' + error.toString()) .setMimeType(ContentService.MimeType.TEXT); } } function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet(); const action = e.parameter.action; const data = JSON.parse(e.postData.contents); try { switch(action) { case 'addProduct': return addRow('products', data); case 'addOrder': return addRow('orders', data); case 'updateProduct': return updateRow('products', data); case 'deleteProduct': return deleteRow('products', data); default: return ContentService.createTextOutput('Invalid action') .setMimeType(ContentService.MimeType.TEXT); } } catch(error) { return ContentService.createTextOutput('Error: ' + error.toString()) .setMimeType(ContentService.MimeType.TEXT); } } function getSheetData(sheetName) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(sheetName); const data = sheet.getDataRange().getValues(); // Skip header row const rows = data.slice(1).map(row => row.map(cell => cell !== null ? cell.toString() : '') ); return ContentService.createTextOutput(JSON.stringify(rows)) .setMimeType(ContentService.MimeType.JSON); } function addRow(sheetName, data) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(sheetName); sheet.appendRow(data); return ContentService.createTextOutput('Success') .setMimeType(ContentService.MimeType.TEXT); } function updateRow(sheetName, data) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(sheetName); const range = sheet.getRange(data.row, 1, 1, data.values.length); range.setValues([data.values]); return ContentService.createTextOutput('Success') .setMimeType(ContentService.MimeType.TEXT); } function deleteRow(sheetName, data) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(sheetName); sheet.deleteRow(data.row); return ContentService.createTextOutput('Success') .setMimeType(ContentService.MimeType.TEXT); }
5

نشر السكريبت كـ Web App

  1. احفظ المشروع (Ctrl+S)
  2. اضغط على Deploy > New Deployment
  3. اختر: Web app
  4. الإعدادات:
  • Description: Admin Dashboard API
  • Execute as: Me
  • Who has access: Anyone
  1. اضغط Deploy
  2. انسخ رابط Web app URL
  3. أضف الرابط في لوحة التحكم
